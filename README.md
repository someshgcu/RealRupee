# RealRupee - Real Estate PWA

A modern real estate platform for buying and selling legally validated properties.

## Tech Stack

- **React** + **TypeScript** with Vite
- **Tailwind CSS** for styling
- **React Router** for navigation
- **Tanstack React Query** for data management
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

The app will start at `http://localhost:8080`.

## Project Structure

- `src/pages/` - Page components (Home, Dashboard, PostProperty, Services, etc.)
- `src/components/` - Reusable components (Navbar, Footer, BottomNav, PropertyCard, etc.)
- `src/data/` - Mock data and type definitions
- `src/context/` - Auth context provider
